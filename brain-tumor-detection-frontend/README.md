# Brain Tumor Detection System

This is a React.js frontend application for the Brain Tumor Detection System. The application allows users to upload MRI scans, analyze them, and view the results of the prediction.

## Features

- **Home Page**: Provides an overview of the application and an option to upload MRI scans.
- **Upload & Prediction Page**: Users can upload MRI images, preview them, and initiate the analysis.
- **Results Page**: Displays the prediction results, including whether a tumor is detected and the confidence score.

## Technologies Used

- React.js
- TypeScript
- Axios for API calls
- CSS for styling

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd brain-tumor-detection-frontend
   ```

3. Install the dependencies:
   ```
   npm installn
   ```

4. Create a `.env` file in the root directory and add the backend API URL:
   ```
   REACT_APP_API_URL=<your-backend-api-url>
   ```

5. Start the development server:
   ```
   npm start
   ```

6. Open your browser and go to `http://localhost:3000` to view the application.

## Usage

- Navigate to the Home Page to learn about the application.
- Click on the "Upload MRI Scan" button to go to the Upload Page.
- Upload an MRI image and click "Analyze Image" to get the prediction.
- View the results on the Results Page.
