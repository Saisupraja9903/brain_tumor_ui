# Brain Tumor Detection Project

---

## 1. Title Slide
**Brain Tumor Detection Using Deep Learning**

---

## 2. Project Briefing
- Objective: Develop an automated system to detect brain tumors from MRI images using deep learning.
- Motivation: Early detection improves treatment outcomes and saves lives.
- Technologies: Python (backend), React/TypeScript (frontend), Deep Learning (CNN models).

---

## 3. Design of the Solution
- **Architecture:**
  - Frontend: React app for image upload, results display, and navigation.
  - Backend: Python Flask API for image processing, model inference, and result delivery.
  - Model: Convolutional Neural Network trained on MRI datasets.
- **Workflow:**
  1. User uploads MRI image via frontend.
  2. Image sent to backend for processing.
  3. Model predicts tumor presence and type.
  4. Results returned and displayed to user.
- **Key Components:**
  - Frontend: UploadPage, ResultsPage, Navigation, Header, Footer.
  - Backend: main.py (API), models.py (model logic), utils.py (helpers).
